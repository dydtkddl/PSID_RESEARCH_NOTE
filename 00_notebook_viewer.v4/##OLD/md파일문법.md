# Markdown 핵심 문법 가이드 (연구노트용)

Markdown(MD)은 텍스트 기반의 경량 마크업 언어로, 연구노트·실험일지·문헌정리·회의록 등을 구조적으로 작성하기에 최적화되어 있다. 아래는 실험노트나 연구정리 작성 시 가장 자주 쓰는 문법만 추려 정리한 실용 가이드이다.

---

## 1. 제목 (Headings)

```md
# H1 제목
## H2 제목
### H3 제목
#### H4 제목
```

연구노트에서는 보통 H1은 문서 제목, H2는 주요 섹션, H3는 하위 항목으로 사용한다.

---

## 2. 목록 (Lists)

### ● 순서 없는 목록

```md
- 항목 1
- 항목 2
  - 하위 항목
    - 더 하위
```

### ● 순서 있는 목록

```md
1. 첫 번째
2. 두 번째
3. 세 번째
```

---

## 3. 강조 (Bold / Italic)

```md
**굵게(bold)**
*기울임(italic)*
***굵게+기울임***
```

연구노트에서는 용어 정의 시 **굵게**, 가설·주요 포인트 강조 시 *기울임*을 사용하면 좋다.

---

## 4. 코드 블록 (Code Blocks)

````md
```python
def run():
    print("hello")
````

````

언어나 형식을 명시하면 문법 하이라이트가 적용된다.

---

## 5. 인라인 코드 (Inline Code)

```md
`pip install package`
````

명령어나 변수 이름, 설정값 등을 문장 안에서 표현할 때 사용.

---

## 6. 수식 (MathJax)

블록 수식:

```md
$$
E = mc^2
$$
```

인라인 수식:

```md
$\alpha + \beta = \gamma$
```

(※ GitHub/GitLab/Obsidian/VSCode 등 대부분 지원)

---

## 7. 인용문 (Blockquote)

```md
> 여기 인용문 또는 요약문을 작성한다.
>> 중첩 인용 가능
```

문헌 리뷰, 논문 구절 요약, 회의 발언 기록에 자주 씀.

---

## 8. 수평선 (구분선)

```md
---
```

문서 내 섹션을 시각적으로 명확히 구분할 때 사용.

---

## 9. 표 (Tables)

```md
| 항목 | 값 | 비고 |
|------|----|------|
| A    | 10 | 예시 |
| B    | 20 | 예시 |
```

복잡한 실험 조건표, 파라미터 테이블, 비교 분석 등에 유용함.

---

## 10. 링크 (Links)

```md
[링크 텍스트](https://example.com)
```

연구논문 URL, 데이터 파일 링크 등에 활용.

---

## 11. 이미지 (Images)

```md
![설명](./images/plot1.png)
```

로컬 이미지나 실험 결과 그래프 첨부 시 사용.

---

## 12. 체크박스 (Task List)

```md
- [ ] 할 일
- [x] 완료된 일
```

TODO 정리, 실험 단계 체크리스트 등에 매우 유용함.

---

## 13. 강조 박스 (주의/노트/요약)

GitHub 스타일:

```md
> **Note**
> 중요한 참고 내용

> **Warning**
> 주의해야 할 내용
```

연구노트 요약 및 프로토콜 정리에 적합.

---

## 14. 접기 (Fold / Details)

GitHub / Obsidian / 일부 렌더러 지원.

```md
<details>
<summary>펼치기 / 접기</summary>

여기에 숨길 내용 작성

</details>
```

많은 이미지나 로그를 숨겨 깔끔한 노트를 만드는 데 좋다.

---

## 15. 파일/폴더 구조 표현 (Tree 구조)

```md
```

project/
├── data/
├── src/
└── results/

```
```

연구 프로젝트 디렉토리 구조를 기록할 때 유용하다.

---

## 16. YAML Front Matter (선택)

Obsidian 또는 정적 사이트 생성기 사용 시 메타데이터 기록 가능.

```md
---
title: "Oligomer Simulation Log"
date: 2025-12-10
tags: [MD, Oligomer, Simulation]
---
```
